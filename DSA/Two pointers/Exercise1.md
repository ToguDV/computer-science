# Exercise 1
## Description
Given a **sorted array** of integers and a **target value**, return the **indices** of two elements whose sum is exactly equal to the target.  
If no such pair exists, indicate so.

### Example
**Input:**  
nums = [1, 2, 4, 7, 11, 15]  
target = 15  

**Output:**  
[1, 4]  
(because nums[1] + nums[4] = 2 + 13 = 15)
